# Rick And Morty API - GraphQL

## :gear: Setup
1. Install npm
```
npm install
```
2. Install Apollo
```
npm install @apollo/client
```

3. Install GraphQL
```
npm install graphql
```

4. Start the app
```
npm start
```

## :wrench: Built with

- [Apollo](https://www.apollographql.com/docs/)
- [GraphQL](https://graphql.org/learn/)
- [Rick And Morty GraphQL](https://rickandmortyapi.com/documentation/#graphql)
